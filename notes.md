# Ziel

In diesem Repo möchte ich meine eigenen Skills speichern. Das dient dazu dass ich meine eigenen Skills einfach auf mehreren Computer installieren kann. Es dient auch dazu dass ich meine Skills mit anderen teilen kann. Es dient auch dazu dass ich meine Skills einfach aktualisieren kann.

# Inspiration
https://github.com/mattpocock/skills

In diesem Repo werden die skills über skills.sh installiert mit "npx skills@latest add mattpocock/skills"

# Aufgabe
Konfigurere dieses Repo so das ich meine eigenen Skills mit npx skills@latest add ratfive/SKILLS installiert werden können.
